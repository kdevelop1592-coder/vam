# 뱀서라이크 게임 구현 계획서

## 프로젝트 개요
- **플랫폼**: 웹 (리액트)
- **목표**: 뱀파이어 서바이버 스타일의 간단한 생존 게임
- **난이도**: 초보자 친화적, 단계별 구현

---

## 1단계: 기본 프로토타입 (MVP)

### 핵심 기능
- 플레이어 캐릭터 (화면 중앙에 고정 또는 키보드로 이동)
- 적 캐릭터 (플레이어를 향해 천천히 이동)
- 자동 공격 (플레이어 주변에서 발사되는 투사체)
- 충돌 감지 (적과 투사체가 닿으면 적 제거)
- 체력 시스템 (적과 닿으면 체력 감소)

### 기술 구현
- `useState`로 플레이어 위치, 체력, 적 목록 관리
- `useEffect` + `setInterval`로 게임 루프 구현
- CSS `position: absolute`로 캐릭터 배치
- 간단한 div/img 태그로 캐릭터 표현

---

## 2단계: 경험치 & 레벨업 시스템

### 추가 기능
- 적 처치 시 경험치 획득
- 경험치 바 표시
- 레벨업 시 게임 일시정지
- 업그레이드 선택 화면 (3가지 옵션 중 1개 선택)

### 업그레이드 예시
- 공격 속도 증가
- 투사체 개수 증가
- 이동 속도 증가
- 체력 회복

---

## 3단계: 게임 시스템 확장

### 추가 요소
- 타이머 (게임 시작 후 경과 시간 표시)
- 적 스폰 시스템 (시간이 지날수록 더 많이, 더 강하게)
- 다양한 적 타입 (빠른 적, 체력 높은 적 등)
- 추가 무기 시스템 (레벨업 시 새 무기 획득 가능)

---

## 4단계: UI/UX 개선

### 디테일 추가
- 게임 오버 화면 (재시작 버튼)
- 승리 조건 (15분 생존 등)
- 파티클 효과 (적 처치 시, 레벨업 시)
- 사운드 효과 및 BGM
- 캐릭터 선택 화면

---

## 5단계: 최적화 & 고급 기능

### 성능 개선
- 적이 많아지면 Canvas로 전환 검토
- 오프스크린 적 제거
- 메모이제이션 적용

### 추가 콘텐츠
- 다양한 캐릭터 (각자 고유 능력)
- 업적 시스템
- 로컬 저장소로 최고 기록 저장
- 다양한 맵/스테이지

---

## 현재 구현 우선순위

**지금 바로 만들 것:**
1. 플레이어 표시 (화면 중앙의 네모 또는 이미지)
2. 키보드 입력으로 이동
3. 적 1마리 생성 및 플레이어 추적
4. 자동 공격 1개 발사

**다음 단계:**
5. 적 여러 마리 스폰
6. 충돌 감지 및 제거
7. 체력 시스템

---

## 개발 팁

- 한 번에 하나의 기능만 추가하고 테스트
- 각 단계마다 AI에게 코드 설명 요청
- 막히면 더 작은 단위로 쪼개기
- 일단 작동하게 만들고, 나중에 예쁘게 만들기

## 예상 개발 시간

- 2단계 (레벨업 시스템): 2-3시간
- 3단계 (시스템 확장): 3-5시간
- 4-5단계 (개선): 자유롭게

**총 예상 시간**: 10-20시간 (초보자 기준)
